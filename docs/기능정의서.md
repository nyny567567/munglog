# 기능정의서

## 프로젝트명

munglog

## 프로젝트 소개

반려견 보호자가 자신의 반려견 정보를 등록하고 일상을 기록할 수 있는 반려견 기록 커뮤니티 서비스

---

## 회원

### 회원가입

* 이메일, 비밀번호, 닉네임으로 가입
* 이메일 중복 불가

### 로그인

* 이메일, 비밀번호 로그인

### 로그아웃

* 로그인 상태 해제

---

## 반려견

### 반려견 등록

* 이름
* 생일
* 견종
* 성별
* 소개글

입력 가능

### 반려견 조회

* 등록한 반려견 정보 조회

### 반려견 수정

* 등록한 정보 수정 가능

### 반려견 삭제

* 등록한 반려견 삭제 가능

---

## 일기

### 일기 작성

* 제목 입력
* 내용 입력
* 작성 날짜 저장

### 이미지 업로드

* 최대 5장 업로드
* jpg, jpeg, png 파일 지원

### 일기 조회

* 목록 조회
* 상세 조회

### 일기 수정

* 작성자만 수정 가능

### 일기 삭제

* 작성자만 삭제 가능

---

## 게시판

### 게시글 작성

* 제목 입력
* 내용 입력
* 작성자 지역 정보 저장

### 게시글 조회

* 게시글 목록 조회
* 게시글 상세 조회
* 지역별 게시글 조회

### 게시글 수정

* 작성자만 수정 가능

### 게시글 삭제

* 작성자만 삭제 가능

---

## 댓글

### 댓글 작성

* 게시글에 댓글 작성

### 댓글 삭제

* 작성자만 삭제 가능

---

## 1차 개발 범위

* 회원가입
* 로그인
* 반려견 등록/수정/삭제
* 일기 작성/조회/수정/삭제
* 일기 이미지 최대 5장 업로드
* 게시글 작성/조회/수정/삭제
* 댓글 작성/삭제

## 추후 확장 예정

* 대표 이미지 선택
* 이미지 순서 변경
* 이미지 개별 삭제
* 채팅
* 실시간 알림
* 모바일 앱
* 공동구매
* 지도 기반 기능
* AI 기반 일기 분석 및 성향 맞춤 메이트 추천 기능
