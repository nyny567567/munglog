# 기능정의서

## 프로젝트명

munglog

## 프로젝트 소개

반려견 보호자가 자신의 반려견 정보를 등록하고 일상을 기록할 수 있는 반려견 기록 커뮤니티 서비스

---

## 회원

### 회원가입

* 이메일, 비밀번호, 닉네임으로 가입
* 이메일 중복 불가

### 로그인

* 이메일, 비밀번호 로그인

### 로그아웃

* 로그인 상태 해제

---

## 반려견

### 반려견 등록

* 이름
* 생일
* 견종
* 성별
* 소개글

입력 가능

### 반려견 조회

* 등록한 반려견 정보 조회

### 반려견 수정

* 등록한 정보 수정 가능

### 반려견 삭제

* 등록한 반려견 삭제 가능

## 일기, 지역 피드 및 댓글

### 구조 및 서비스 방향
본 서비스는 별도의 게시판 엔티티를 생성하지 않으며, '일기' 데이터를 활용하여 개인 블로그와 지역 피드 서비스를 통합 제공합니다.

### 일기 작성
* 제목, 내용 입력 및 작성 날짜 저장
* **이미지 업로드**: 최대 5장 (jpg, jpeg, png)
* **공개 설정**:
    * [전체 공개]: 지역 피드에 자동 노출 (작성자 지역 정보 기반 필터링)
    * [비공개]: 본인만 열람 가능
* **댓글 설정**: 댓글 허용/차단 여부 설정

### 일기 조회 및 피드
* **내 블로그**: 작성한 모든 일기 목록/상세 조회
* **동네 피드**: [전체 공개]로 설정된 타인의 일기를 지역별로 모아보기
* **조회수**: 일기 상세 조회 시 조회수 카운트 기능

### 일기 수정 및 삭제
* 작성자 본인만 수행 가능

### 댓글 기능
* **작성 조건**: 본인의 일기 혹은 타인의 '전체 공개' 일기 중 '댓글 허용' 설정이 된 일기에만 작성 가능
* **댓글 삭제**: 작성자 본인 또는 해당 일기의 주인(작성자)만 삭제 가능

## 1차 개발 범위

* 회원가입 및 로그인
* 반려견 등록/수정/삭제
* 일기 작성(공개 범위/댓글 설정 포함)/조회/수정/삭제
* 일기 이미지 최대 5장 업로드
* 지역 기반 일기 피드 노출 및 조회
* 일기(지역 피드) 댓글 작성 및 삭제
